FLUJO RAPIDO (APRENDIZAJE)

1) Ejecutar bot para recolectar datos
   - python main.py

2) Exportar metricas de la sesion actual
   - .venv\Scripts\python.exe aprendizaje\scripts\exportar_metricas_aprendizaje.py

3) Exportar reporte profundo de caja negra
   - .venv\Scripts\python.exe lab\full_session_review.py
   - .venv\Scripts\python.exe lab\black_box_stratb.py

4) Exportar velas para calculos offline
   - .venv\Scripts\python.exe lab\dump_candidate_candles.py

5) Cerrar y archivar sesion
   - powershell -ExecutionPolicy Bypass -File aprendizaje\scripts\cerrar_sesion_aprendizaje.ps1

SALIDAS ESPERADAS
- aprendizaje/reportes/metricas_YYYYMMDD_HHMMSS.csv
- aprendizaje/sesiones/YYYYMMDD_HHMM/ (DB + log + sesion.json)
